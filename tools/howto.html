What Is How-To Schema?

How-To Schema is a structured data type that describes a process or procedure with step-by-step instructions. It includes properties for the overall task, individual steps, required tools, necessary supplies, and estimated total time. Google displays this information as rich how-to results with expandable steps, images, and time estimates.

Features

  • Task name and description for the overall procedure
  • Step-by-step instructions with position-based ordering
  • Required tools list with HowToTool markup
  • Necessary supplies list with HowToSupply markup
  • Total time specification using ISO duration format
  • Image URL support for visual how-to results
  • Valid JSON-LD output ready to implement

Benefits

How-To Rich Results

Google displays your steps as an interactive, expandable guide directly in search results, increasing visibility and engagement.

Voice Search Optimization

Virtual assistants can read your how-to instructions aloud, making your content accessible through smart speakers and voice search.

Higher Engagement

Users who see clear, step-by-step instructions in search results are more likely to click through and complete the process on your site.

How It Works

  1. Enter the task name and a brief description of the procedure.
  2. Add an optional image URL for visual how-to results.
  3. Enter estimated total time in ISO format (e.g., PT30M for 30 minutes).
  4. List required tools (one per line) and necessary supplies (one per line).
  5. Write step-by-step instructions (one per line) in chronological order.
  6. Copy the generated JSON-LD to your tutorial or guide page.

Frequently Asked Questions

How many steps should a How-To have?

Google recommends at least 2 steps. More detailed procedures can have many steps, but keep each step clear and actionable.

What is ISO duration format?

Format like PT15M for 15 minutes, PT2H for 2 hours, or PT1H30M for 1 hour 30 minutes.

Can I include images for individual steps?

Yes, you can add image URLs to individual HowToStep items for enhanced visual instructions.

Is How-To Schema the same as Recipe Schema?

No. Recipe Schema is for food and drink recipes. How-To Schema is for general procedures and instructions.

Do I need tools and supplies for every How-To?

Only include tools and supplies that are relevant to your procedure. They are optional but recommended for completeness.
