What Is ImageObject Schema?

ImageObject Schema is a Schema.org type that describes an image or photograph. It includes properties such as name, description, content URL, author, publication date, license, copyright holder, and credit text. Google uses this markup to display images with enhanced metadata in search results, helping users find properly attributed visual content.

For stock photography sites, portfolio websites, and news media, ImageObject Schema ensures your images are correctly attributed and discoverable.

Frequently Asked Questions

Is ImageObject Schema required for Google Images?

It is not required, but it significantly improves how your images are displayed and attributed in Google Image Search.

What license URLs should I use?

Common licenses include Creative Commons URLs (e.g., https://creativecommons.org/licenses/by/4.0/) or your own custom license page.

Can I use ImageObject Schema for multiple images on one page?

Yes, create separate ImageObject entries for each unique image on the page.

What is the difference between contentUrl and url?

contentUrl points directly to the image file, while url points to the webpage displaying the image.
