What Is Dataset Schema?

Dataset Schema is a structured data type that describes a collection of data points or files available for access or download. It includes properties for name, description, image, URL, creator, license, temporal and spatial coverage, keywords, and publication dates. Google uses this markup to display dataset rich results that help researchers discover and cite your data.

Features

  • Dataset name and detailed description
  • Creator and publisher attribution
  • License URL support for usage rights
  • Temporal coverage for time-based data
  • Spatial coverage for geographic data
  • Keyword support for search categorization
  • Publication and modification dates
  • Image and URL for dataset access

Frequently Asked Questions

Is Dataset Schema required for Google Dataset Search?

Yes, Dataset Schema is required for inclusion in Google Dataset Search results.

What is temporal coverage?

Temporal coverage describes the time period the dataset covers, such as 2020-2024. Use ISO 8601 date format.

What is spatial coverage?

Spatial coverage describes the geographic area the dataset covers, such as United States or New York City.

Do I need a license for my dataset?

Including a license is strongly recommended so users know how they can use and cite your data.

Can I update dataset schema after publication?

Yes, update the dateModified field whenever you add, update, or correct your dataset.
