What Is ItemList Schema?

ItemList Schema is a Schema.org type that describes an ordered or unordered list of items. Each item in the list is marked up as a ListItem with a position, name, and URL. Google uses this structured data to display list rich results, carousels, and featured list snippets that make your content stand out in search.

List rich results are particularly effective for rankings, recommendations, top picks, and curated collections where users want to browse multiple options.

Frequently Asked Questions

How many items can I include?

This generator supports up to 5 list items. Google recommends keeping lists focused and concise.

Does the order of items matter?

Yes, position numbers should reflect the intended order, especially for ranked lists.

Can I use ItemList for product lists?

Yes, Product schema can be used within ListItem elements for enhanced product list results.

Is ItemList the same as Carousel?

Both use ItemList schema, but carousels require additional image and type properties for each item.
