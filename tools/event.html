Get More Attendees With Event Schema in Search

When you organize an event, you want it to be visible to everyone searching for things to do. Event Schema makes your events appear directly in Google Search with dates, times, locations, and ticket information. It is the most effective way to ensure your events are discovered by the right audience.

Our Event Schema Generator creates JSON-LD markup for conferences, concerts, webinars, workshops, and any type of event.

What Is Event Schema?

Event Schema describes an event including its name, description, start and end dates, location, and ticket availability. Google displays this information in rich event results that appear in search and Google Maps, helping potential attendees find and plan to attend your events.

Features

  • Event name, description, and image
  • Start and end date/time with ISO format
  • Location and venue information
  • Ticket pricing and availability
  • Currency support for international events
  • Valid JSON-LD output

Benefits

Event Rich Results

Events with schema appear with dates and locations directly in search, increasing visibility and attendance.

Google Maps Integration

Event locations appear in Google Maps searches for things to do in an area.

How It Works

  1. Enter the event name, description, and image URL.
  2. Set the start and end dates and times.
  3. Add the venue or location name.
  4. Include the event URL and optional ticket pricing.
  5. Copy the generated JSON-LD to your event page.

Frequently Asked Questions

Can I use Event Schema for recurring events?

Yes, mark up each occurrence separately or use the eventSchedule property for recurring events.

Does Event Schema work for online events?

Yes, use VirtualLocation for online events and Place for physical venues.
