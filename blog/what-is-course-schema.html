What is Course Schema?

June 2026 · 483 words

Course schema is a type of structured data for describing educational courses. It provides properties for course name, description, provider, educational level, and syllabus. When Google detects valid Course schema, it can display your courses in rich results with course details, ratings, and provider information. Course schema is essential for educational institutions, online learning platforms, and any organization offering structured educational content.

Key Properties of Course Schema

Course schema includes properties for comprehensive course descriptions. The name property is the course title. The description property summarizes what the course covers. The provider property identifies the educational organization using Organization schema. The educationalCredentialAwarded property describes the certificate or degree earned. The courseCode property provides the institutional course identifier. The hasCourseInstance property links to specific offerings using CourseInstance schema with dates, locations, and enrollment information.

Course Rich Results

Course schema enables rich results that display course name, provider, rating, and pricing directly in search results. These rich results can include review stars and educational level information. For online courses, the rich result can include a direct link to enroll. For in-person courses, location and schedule information can be displayed. Course rich results help students find and compare educational opportunities directly from search results.

CourseInstance Schema

The CourseInstance schema works alongside Course to describe specific offerings of a course. Each CourseInstance can have its own startDate, endDate, location, and offers properties. This allows you to describe multiple sessions of the same course offered at different times or locations. The courseMode property indicates whether the offering is in-person, online, or mixed. The instructor property identifies who teaches that specific offering.

Educational Level and Audience

The educationalLevel property describes the target audience using terms like Beginner, Intermediate, Advanced, or specific grade levels. The typicalAgeRange property specifies the recommended age for learners. The audience property using Audience schema can provide more specific targeting information. These properties help students find courses appropriate for their skill level and educational background.

Provider and Accreditation

The provider property identifies the organization offering the course using EducationalOrganization schema. Include the provider name, url, and logo. For accredited courses, include accreditation information. The offers property can describe tuition costs, payment plans, and financial aid options. A complete provider profile helps establish credibility and helps students evaluate the quality of the educational offering.

Syllabus and Learning Outcomes

Course schema supports detailed curriculum information. The description property can include an overview of topics covered. The teaches property identifies specific skills or concepts taught. The educationalCredentialAwarded property describes what students earn upon completion. The competencyRequired property lists prerequisites. Comprehensive curriculum information helps students understand what they will learn and whether the course meets their educational goals.

Conclusion

Course schema is essential for educational institutions and online learning platforms. It enables course rich results that help students discover and compare educational opportunities.
