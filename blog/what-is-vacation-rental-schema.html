What is Vacation Rental Schema?

June 2026 · 420 words

VacationRental schema is a type of structured data for describing short-term rental properties such as apartments, houses, and villas available for vacation stays. It provides properties for property name, description, location, amenities, pricing, and availability. VacationRental schema helps your rental property appear in Google Search with detailed information that helps travelers find and book accommodations.

Key Properties of VacationRental Schema

VacationRental schema includes properties for comprehensive property descriptions. The name property identifies the rental property. The description summarizes what the property offers. The numberOfBedrooms and numberOfBathrooms describe the property layout. The occupancy property indicates the maximum number of guests. The amenityFeature lists property amenities using LocationFeatureSpecification schema. The petsAllowed property indicates whether pets are permitted. The checkinTime and checkoutTime specify the rental schedule.

Pricing and Availability

The offers property using Offer schema describes pricing and availability for your rental property. Include properties for price, priceCurrency, and availability. The availableAtOrFrom and availabilityEnds properties define the rental period. For seasonal pricing, include multiple Offer instances with different price specifications. The priceRange property provides a general indication of cost level. Clear pricing information helps travelers compare properties and make informed booking decisions.

Location and Nearby Attractions

The address property using PostalAddress schema provides the property location. The geo property with GeoCoordinates provides map coordinates. The containedInPlace property can link to the larger complex or neighborhood. For nearby attractions, use the nearbyPlace property with Place schema describing restaurants, beaches, landmarks, or other points of interest. Location information is critical for travelers searching for rentals in specific areas.

Amenities and Features

The amenityFeature property uses LocationFeatureSpecification schema to describe property amenities. Common amenities include WiFi, parking, pool, air conditioning, heating, washer, dryer, kitchen, and gym access. The floorSize property describes the property square footage. The numberOfRooms provides the total room count. Comprehensive amenity information helps travelers find properties that meet their specific needs and preferences.

Reviews and Ratings

The aggregateRating property displays average guest ratings in search results. The review property with Review schema marks up individual guest reviews. Star ratings and review counts provide valuable social proof that can significantly influence booking decisions. Ensure that reviews are from genuine guests and are visible on your property page. Positive reviews with detailed feedback help your property stand out in search results.

Conclusion

VacationRental schema is essential for property owners and rental platforms listing short-term accommodations. It helps your properties appear in search results with detailed information that attracts travelers.
