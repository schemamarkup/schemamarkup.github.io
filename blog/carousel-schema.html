What is Carousel Schema?

June 2026 · 392 words

Carousel schema markup allows you to display a scrollable list of items directly in Google search results. When implemented correctly, your content appears as a horizontal carousel that users can swipe or click through. Carousels are visually prominent and can display multiple items from your site in a single search result. They are particularly effective for recipe collections, movie lists, album catalogs, and other content that benefits from browsing multiple items.

How Carousel Schema Works

Carousel schema uses the ItemList type with a special carousel configuration. Each item in the list is represented using the appropriate schema type such as Recipe, Movie, or Article. The list items are ordered and displayed as a scrollable carousel. Google supports several carousel types including course carousels, recipe carousels, and movie carousels. The specific requirements vary by carousel type, but all require properly formatted schema markup for each item in the list.

Types of Carousels

Google supports multiple carousel types. The recipe carousel displays recipes with photos, ratings, and cooking times. The course carousel shows educational courses with provider and rating information. The movie carousel displays films with ratings, release dates, and cast information. The restaurant carousel shows local dining options with cuisine types and prices. Each carousel type requires the corresponding schema type for individual items within the list.

Carousel Requirements

To be eligible for a carousel rich result, your page must list multiple items of the same type. Each item must have complete, valid schema markup. The carousel requires at least two items, though more items provide a better user experience. All items should be from the same website and relate to a common theme or category. Google's guidelines require that the content is visible and accessible on the page, not hidden from users.

ItemList Structure

The ItemList type organizes items for carousel display. The itemListElement property contains the ordered list of items. Each item should be a ListItem with position and item properties. The position starts at 1 and increments sequentially. The item property contains the full schema markup for each individual item using the appropriate type. The number of items property provides the total count for display purposes.

Conclusion

Carousel schema is a powerful way to showcase multiple items from your website in a single, visually prominent search result. Our Schema Markup Generator supports the schema types needed to create carousel-eligible markup.
