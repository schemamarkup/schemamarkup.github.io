What is HowTo Schema?

June 2026 · 545 words

HowTo schema is a type of structured data that allows you to mark up instructional content with step-by-step directions. When Google detects valid HowTo schema, it can display your instructions directly in search results as a rich result with expandable steps, images, and timing. HowTo rich results are highly engaging and can significantly increase click-through rates for instructional content such as recipes, DIY guides, tutorials, and assembly instructions.

How HowTo Schema Works

HowTo schema uses the HowTo type which contains a list of steps using HowToStep or HowToSection schema. Each step can include text instructions, images, videos, and timing information. The overall HowTo object includes properties for the total time required, the tools and materials needed, and a summary of the process. Google displays this information in an expandable rich result format that users can interact with directly from the search results page.

Key Properties of HowTo Schema

HowTo schema includes important properties for describing instructional content. The name property identifies the overall process. The description property summarizes what the instructions cover. The totalTime property indicates the total duration in ISO 8601 format. The tool and supply properties list the tools and materials needed. The step property contains the individual HowToStep objects. Each step can have position, text, image, video, and duration properties for comprehensive instructions.

HowTo Schema vs. Recipe Schema

HowTo schema is more general than Recipe schema, which is a specialized subtype of HowTo for food recipes. Recipe schema includes food-specific properties like ingredients, nutrition, and cooking methods. HowTo schema is suitable for all types of instructional content including repair guides, assembly instructions, craft tutorials, and fitness routines. Choose HowTo for general instructions and Recipe specifically for food and cooking content.

Steps with Images

One of the most powerful features of HowTo schema is the ability to include images for each step. The image property within HowToStep allows you to provide a visual guide alongside the text instructions. Images make instructions easier to follow and significantly improve user engagement. Google may display step images directly in the rich result, creating a more visually appealing and informative search listing. Use clear, high-resolution images that show exactly what each step accomplishes.

Video Integration

HowTo schema supports video content through the video property at both the HowTo and HowToStep levels. A video at the HowTo level provides an overview of the entire process. Videos at the step level demonstrate specific techniques or actions. Video instructions are particularly effective for complex procedures where visual demonstration is more helpful than text descriptions. Including video with your HowTo markup can improve engagement and completion rates.

Tools and Materials

The tool and supply properties in HowTo schema allow you to list the items needed to complete the instructions. The tool property lists equipment, tools, or appliances required. The supply property lists consumable materials or ingredients. Each item can be described using Thing schema with name, image, and url properties. Listing required items upfront helps users prepare before starting and reduces frustration from missing materials mid-process.

Conclusion

HowTo schema is essential for any website with instructional content. It enables engaging step-by-step rich results that keep users on your page and improve search visibility.
