What is ImageObject Schema?

June 2026 · 393 words

ImageObject schema is a type of structured data that describes images on your website. It provides properties for image caption, description, author, license, and more. While images are often described within other schema types like Product or Article, standalone ImageObject markup can help search engines understand your images better and display them in image search results and rich results.

Key Properties of ImageObject Schema

ImageObject schema includes properties for comprehensive image descriptions. The name property provides a title for the image. The description property offers additional context. The contentUrl property links to the image file. The caption property provides a visible caption. The author property credits the image creator. The license property specifies usage rights. The thumbnail property links to a smaller version of the image. These properties help search engines understand what your images depict and how they can be used.

Image Object in Other Schemas

ImageObject is most commonly used as a nested object within other schema types. Product schema uses the image property to provide product photos. Article schema uses image for featured images. Recipe schema uses image for food photography. Local Business schema uses image for business photos. When you include complete ImageObject markup within these parent schemas, Google can display your images in rich results with proper attribution and context.

Image Licensing and Rights

The license and copyrightHolder properties in ImageObject schema describe usage rights for your images. These properties are particularly important for stock photography, creative portfolios, and any website where image licensing matters. The acquireLicensePage property links to a page where users can purchase or obtain a license. Clear licensing information helps protect your image rights and can increase revenue from image licensing.

Image SEO Benefits

ImageObject schema can improve your images visibility in Google Image Search. While Google primarily uses file names, alt text, and surrounding content for image indexing, schema markup provides additional signals about image content and context. Images with complete schema markup may be more likely to appear in image-rich results and knowledge panels. For ecommerce sites, product images with proper schema are essential for product rich results.

Conclusion

ImageObject schema helps search engines understand your images and can improve their visibility in search results. It works both as standalone markup and as a nested component within other schema types. Our Schema Markup Generator includes ImageObject support within all relevant schema types.
